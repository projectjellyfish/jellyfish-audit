Jellyfish Audit
=======
Adds audit support to [Project Jellyfish] (https://github.com/projectjellyfish/api).

#### Installation
Include in Gemfile:
```
  gem 'jellyfish-audit'
```
or download from GitHub and reference locally:
```
gem 'jellyfish-audit', path: '../jellyfish-audit'
```
